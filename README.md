# MCP-Based Enterprise AI Customer Support Agent

AI-powered enterprise customer support agent built using Python, FastAPI, LLM APIs, and MCP-style tool integration.

This project simulates a modern GenAI support system that can understand customer issues, classify support requests, search internal knowledge documents, check customer information, create support tickets, and generate AI-assisted responses.

## Features

- Accepts customer support messages as input
- Classifies support issues using an LLM
- Uses MCP-style tools for external system actions
- Simulates Slack, Jira, Confluence, and Salesforce integrations
- Generates support responses using LLM APIs
- Demonstrates agentic AI workflow automation

## Tech Stack

- Python
- FastAPI
- OpenAI / Anthropic / Hugging Face APIs
- MCP-style tool layer
- RAG
- Slack API
- Jira API
- Confluence API
- Salesforce API

## Project Goal

The goal of this project is to demonstrate real-world Generative AI engineering skills, including Python backend development, LLM API integration, agent workflows, tool usage, and enterprise automation.
